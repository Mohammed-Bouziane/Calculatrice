# Calculator

This is a simple calculator implemented in HTML, CSS, and JavaScript. It allows users to perform basic arithmetic calculations.

# Usage

1. Open the index.html file in a web browser.
2. The calculator interface will be displayed.
3. Click on the buttons to enter numbers and operators.
4. The entered values will be displayed in the input field at the top.
5. To perform a calculation, click the "=" button.
6. The result will be displayed in the input field.
7. To clear the input field, click the "C" button.

# Features

. Addition, subtraction, multiplication, and division operations are supported.
. The calculator handles basic error cases, such as division by zero and syntax errors.
. The calculator has a responsive design and adjusts to different screen sizes.

# Dependencies

No external dependencies are required for this calculator.

# File Structure

. index.html: The main HTML file that contains the calculator interface.
. style.css: The CSS file that defines the styles for the calculator.
. script.js: The JavaScript file that handles the calculator's functionality.
